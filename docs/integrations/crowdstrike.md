## Available actions

Once connected, you can use CrowdStrike actions as steps in your Mindflow flows. Browse the available operations in the flow editor when adding a new step.

Common automation scenarios using CrowdStrike with Mindflow include event-driven workflows, data synchronization, and automated responses to alerts.
