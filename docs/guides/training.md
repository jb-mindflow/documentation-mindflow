# Training

In this section, you will come across different Training programs that will help you onboarding on Mindflow and master the different features.

:::note
![Icon](https://www.notion.so/icons/bell-notification_blue.svg?mode=light)

:::note
This section is currently in progress. Training programs will be released one-by-one. Hang tight!
:::
:::

* * *

-   Stage 1

In this first stage, you will learn the basics of Mindflow: Creating an account, your first API credentials, data transformation, data pills, AI··Tasks, and sending emails!

[![Stage 1 - Onboarding and first Flow](https://www.notion.so/icons/123_blue.svg?mode=light)

Stage 1 - Onboarding and first Flow](/guides/training/stage-1-onboarding-and-first-flow)

* * *

-   Stage 2

In this second stage, you will go beyond linear flows: using For Each loops to process lists of data, empowering AI··Tasks with web search tools, parsing JSON output with JSONata, and delivering automated newsletters straight to your inbox!

[![Stage 2 - Advanced Capabilities](https://www.notion.so/icons/123_blue.svg?mode=light)

Stage 2 - Advanced Capabilities](/guides/training/stage-2-advanced-capabilities)

* * *

-   Stage 3

In this third stage, you will build modular and intelligent workflows: extracting data with JSONPath, routing tickets with internal Dictionaries, analyzing text with AI··Tasks, and orchestrating multi-flow automation using Subflows!

[![Stage 3 -  Subflows, AI Tasks, and Data Manipulation](https://www.notion.so/icons/123_blue.svg?mode=light)

Stage 3 - Subflows, AI Tasks, and Data Manipulation](/guides/training/stage-3-subflows-ai-tasks-and-data-manipulation)

* * *

-   Stage 4

In this fourth stage, you will discover the Model Context Protocol (MCP): setting up a reusable MCP Agent, interacting with the GitHub MCP server to fetch and triage open issues, and combining MCP calls with native Tool Calls to build a production-grade agentic workflow!

[![Stage 4 - MCP Client: Automating GitHub Issue Triage](https://www.notion.so/icons/123_blue.svg?mode=light)

Stage 4 - MCP Client: Automating GitHub Issue Triage](/guides/training/stage-4-mcp-client:-automating-github-issue-triage)
