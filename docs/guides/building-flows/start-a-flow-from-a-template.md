# Start a Flow from a Template

The fastest way to build a flow is to start from one of our pre-built templates. Templates cover common use cases across security, IT operations, and business workflows.

## Choosing a template

Browse the template library by category or search for a specific use case. Each template includes a description, the services it uses, and a preview of the flow structure.

## Customizing your flow

After selecting a template, customize it by connecting your own integrations, adjusting parameters, and modifying the logic to fit your specific requirements.
