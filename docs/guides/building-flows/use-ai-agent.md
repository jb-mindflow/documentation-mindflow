# Use AI Agent

Leverage AI-powered agents within your flows to handle complex decision-making and text processing tasks.

## What are AI agents?

AI agents in Mindflow use large language models to analyze data, make decisions, generate content, and perform natural language processing within your automation flows.

## Adding an AI agent step

Add an AI agent step to your flow, write a prompt that describes what you want the agent to do, and map the input data it should analyze.

## Best practices

Write clear, specific prompts. Provide context and examples when possible. Test with representative data to verify the agent produces the expected output.
