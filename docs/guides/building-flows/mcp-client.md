# MCP Client

Connect Mindflow to Model Context Protocol (MCP) servers to extend AI capabilities with external tools and data sources.

## What is MCP?

The Model Context Protocol (MCP) is an open standard for connecting AI models to external tools and data sources. Mindflow's MCP client allows your AI-powered flows to leverage MCP servers.

## Connecting to MCP servers

Configure MCP server connections in your Mindflow workspace. Once connected, MCP tools become available as steps in your AI-powered flows.

## Supported MCP features

Mindflow supports MCP tool calls, resource access, and prompt templates from connected MCP servers.
