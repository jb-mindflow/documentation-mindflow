# Run a Script

For advanced use cases, execute custom scripts within your flows.

## Supported languages

Mindflow supports custom script execution for scenarios where built-in steps don't cover your specific requirements.

## Writing scripts

Write your script in the built-in editor. Access data from previous steps through the script context and return results for use in subsequent steps.

## Security considerations

Scripts run in a sandboxed environment. Follow security best practices and avoid hardcoding credentials in your scripts.
