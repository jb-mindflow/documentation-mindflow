# AITask: Prompt Guidelines

AITask steps use AI to process data within your flows. Writing effective prompts is key to getting reliable results.

## Prompt structure

A good AITask prompt includes context about the data, a clear instruction, and the expected output format.

## Tips for effective prompts

Be specific about what you want. Provide examples when possible. Specify the output format (JSON, plain text, list, etc.). Include constraints and edge cases.

## Common use cases

AITask is commonly used for text classification, data extraction, summarization, translation, and content generation within automation flows.
